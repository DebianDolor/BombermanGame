# BombermanGame

## Preview
![Bomberman](Bomberman.png)

## Mode
### Play:
Direction: ↑, ↓, ←, → <br/>
Bomb: SPACE

### AI:
Autoplay

## Instruction
ENTER: Move to next stage (after enter portal) <br/>
R: Play again (after enter portal) <br/>
ESC: Exit game to main menu
